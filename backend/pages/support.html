üöÄ Getting Started

How do I start a video chat? +

1. Click "Start Chat" on the homepage
2. Allow camera and microphone permissions
3. Set your preferences (country, gender)
4. Wait to be matched with someone

Do I need to create an account? +

No! Saathi TV is completely anonymous. No registration or personal information required.

üîß Technical Issues

Camera not working? +

1. Check browser permissions (click camera icon in address bar)
2. Close other apps using camera
3. Refresh the page
4. Try a different browser
5. Restart your device

Poor video quality? +

1. Check your internet connection
2. Close other bandwidth-heavy applications
3. Move closer to your router
4. Try switching to a wired connection

No sound or microphone issues? +

1. Check microphone permissions
2. Ensure microphone is not muted
3. Check system volume settings
4. Test with other applications
5. Try using headphones
